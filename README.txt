*** SIMULATEUR ET OPTIMISATEUR DE WARGAME***

Utilisation:


Windows:
	Naviguez dans le dossier "src", et double-cliquez sur le fichier "wargame.py" ci-trouvant à la racine du dossier pour lancer le simulateur.

Linux:
	Ouvrez un terminal dans la racine du dossier, et saisir le suivant:
		python src/wargame.py
		
		
		
		
Description des fichiers:

combat.py: Fonctions de combat utilisés lors du simulation.
main_menu.py: Fonctions pour le menu principale.
modeles.py: Fonctions principales du programme, boucle de programme, etc. Elle contient aussi tous les classes et fonctions principales.
optimisateur.py: Fonctions pour l'optimisateur d'armée.
unites.py: Classes des unités utilisés lors du simulation.
wargame.py: Le ficher "exécuté" pour lancer le simulateur; le point d'entrée du programme.

docs/rapport/rapport.pdf: Le rapport pour le soutenance du projet, écrit et compilé en LaTex.
